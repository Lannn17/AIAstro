# 开放注册 + 用户隔离（2026-03-27）

> 前置条件：后端跑在 8001，前端跑在 5173。需准备两个测试账号 `testuser1` / `testuser2`。

## 1. 注册流程

- [×] 登录按钮 → LoginModal → 出现「登录」和「注册」两个 tab，默认在「登录」tab
- [×] 切到「注册」tab → 显示用户名、密码、确认密码三个输入框
