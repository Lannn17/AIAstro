# 开放注册功能 — 手动测试清单

**日期：** 2026-03-27
**功能：** 开放注册 + 用户隔离 + 管理员可见全部数据（v0.8.0）

---

## 前置条件

- 后端运行中：`cd astrology_api && uvicorn main:app --host 127.0.0.1 --port 8001 --reload`
- 前端运行中：`cd frontend && npm run dev`
- 浏览器打开：`http://localhost:5173`

---

## 1. 注册流程

1. 点击右上角登录按钮，打开 LoginModal
   - 预期：显示「登录」和「注册」两个 tab，默认在「登录」tab

2. 切换到「注册」tab
   - 预期：显示用户名、密码、确认密码三个输入框

3. 测试客户端校验 — 密码不一致
   - 输入：用户名 `testuser1`，密码 `123456`，确认密码 `654321`
   - 点击注册
   - 预期：显示「两次密码不一致」错误，不发请求

4. 测试客户端校验 — 密码过短
   - 输入：用户名 `testuser1`，密码 `123`，确认密码 `123`
   - 点击注册
   - 预期：显示密码长度错误（不少于6位），不发请求

5. 正常注册第一个用户
   - 输入：用户名 `testuser1`，密码 `password123`，确认密码 `password123`
   - 点击注册
   - 预期：注册成功，modal 关闭，右上角显示 `testuser1` 用户名

6. 重复注册相同用户名
   - 退出登录，再次打开注册 tab
   - 输入相同用户名 `testuser1`，密码 `password123`
   - 预期：显示「用户名已存在」错误（409）

7. 注册时使用管理员用户名
   - 尝试注册用户名 = `AUTH_USERNAME`（.env 中的值）
   - 预期：显示「该用户名不可用」错误（400）

---

## 2. 登录流程（已注册用户）

8. 登出后用注册账号登录
   - 使用 `testuser1` / `password123` 登录
   - 预期：登录成功，modal 关闭，显示用户名

9. 用错误密码登录
   - 使用 `testuser1` / `wrongpassword` 登录
   - 预期：显示「用户名或密码错误」（401）

---

## 3. 用户数据隔离

10. 以 `testuser1` 登录，生成并保存一张星盘
    - 路径：星盘 tab → 填写信息 → 计算 → 保存
    - 预期：星盘出现在左侧列表中

11. 注册并登录第二个用户 `testuser2`
    - 预期：侧边栏列表为空（看不到 `testuser1` 的星盘）

12. `testuser2` 保存一张自己的星盘
    - 预期：侧边栏只显示 `testuser2` 自己的星盘

13. 直接访问 `testuser1` 的星盘 ID（通过 API）
    - 以 `testuser2` 身份调用 `GET /api/charts/{testuser1_chart_id}`
    - 预期：返回 403「无权访问此星盘」

---

## 4. 管理员权限

14. 以管理员账号（`AUTH_USERNAME` / `AUTH_PASSWORD`）登录
    - 预期：登录成功

15. 查看星盘列表
    - 预期：`GET /api/charts` 返回所有用户（testuser1、testuser2）的星盘

16. 访问 `testuser1` 的星盘
    - 预期：管理员可正常读取（200 OK）

17. 访问待审核队列
    - 预期：`GET /api/charts/pending` 返回 200（管理员权限正常）

18. 以普通用户访问待审核队列
    - 以 `testuser1` 身份调用 `GET /api/charts/pending`
    - 预期：返回 403

---

## 5. 访客模式（回归）

19. 不登录（访客模式），生成一张星盘
    - 预期：可以计算，可选择保存（进入待审核队列）

20. 访客保存星盘后，以管理员查看待审核队列
    - 预期：访客星盘出现在 `/api/charts/pending`

---

## 6. 注销与状态清理

21. 登出 `testuser1`
    - 预期：右上角显示未登录状态，星盘列表清空

22. 刷新页面后重新登录
    - 预期：登录状态通过 localStorage token 正确恢复，列表重新加载
